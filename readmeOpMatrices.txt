# Matrices 5 operaciones

## Programa que permite realizar 5 operacones con matrices y vectores proporcionados por el usuario. Dichas operaciones son Suma de dos matrices de nxm, Resta de dos matrices de nxm, Multiplicación de matrices de nxm, Multiplicación de un vector de nx1 por una matriz de nxm y Suma de columnas de una matriz.

## Asegúrate de tener lo siguiente instalado en tu sistema:
- Un compilador de C, como GCC (GNU Compiler Collection): Puedes verificar si está instalado ejecutando `gcc --version` y descargarlo si es necesario.
- [Opcional] CMake para la construcción del proyecto.

## Configuración
1. Clona este repositorio: `git clone https://github.com/Cta06/Matrices_5_Operaciones.git`
2. Navega al directorio del proyecto: `Matrices_5_Operaciones`

## Compilación
### Usando GCC (GNU Compiler Collection)
Ejecuta el siguiente comando para compilar tu código en C:
gcc archivo.c -o mi_programa


### Usando CMake [Opcional]
Si utilizas CMake para administrar tu proyecto, sigue estos pasos:
1. Crea un directorio `build` en la raíz de tu proyecto: `mkdir build && cd build`
2. Genera los archivos de construcción con CMake: `cmake ..`
3. Compila el proyecto: `make`

## Uso
1. Ejecuta tu programa compilado: `./mi_programa` (si usaste GCC) o navega al directorio de construcción y ejecuta el programa desde allí (si usaste CMake).
